# Laboratory-6

1. Write a program that uses the Scanner class and the newer IO class to read a user's name and age, then print a personalized greeting.

2. Write a program to display the Fibonacci series up to N terms.

3. Write a program that inputs 5 numbers and stores them in an array, then calculates and prints their sum and average.

4. Write a program to check if an input string is a palindrome (reads the same forwards and backward).

5. Create a Rectangle class with length and width as fields and a method calculateArea(). Demonstrate creating and using objects of this class.

6. Setup: Create an ArrayList of Product objects, where Product has fields like name (String) and price (double).
Activity:
Use the stream() method on the list.
Apply the filter() intermediate operation using a lambda expression to select all products whose price is greater than a specific value provided by the user (e.g., 50.00, 65.00, 30.10).
Use the count() terminal operation to print the total number of expensive products found.
